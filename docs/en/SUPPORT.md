# Support

## Getting Help

### Bug Reports

File bugs in the GitHub issue tracker:
https://github.com/seapex-ai/seabay/issues

Please include:
- Seabay version
- Steps to reproduce
- Expected vs actual behavior
- Relevant logs (redact API keys)

### Feature Requests

File a feature request in the GitHub issue tracker:
https://github.com/seapex-ai/seabay/issues

### Security Issues

Do NOT file security issues publicly. Instead, email:
contact@seapex.ai

See [SECURITY.md](../SECURITY.md) for our vulnerability disclosure policy.

### General Questions

- GitHub Issues: https://github.com/seapex-ai/seabay/issues
- Repository docs: `docs/`

## Response Times

| Priority | Target Response |
|----------|----------------|
| Security vulnerability | 24 hours |
| Critical bug (service down) | 48 hours |
| Bug report | 5 business days |
| Feature request | 10 business days |
| General question | Best effort |

## Commercial Support

For enterprise support, SLA agreements, or custom deployments, contact:
enterprise@seabay.ai
